Contains the Log Wrapper implementation for the JCR API. The Log Wrapper is used to log application method calls to a JCR implementation, for example Jackrabbit. The generated log contains all method calls to all methods of the JCR API, together with the parameter values used. The file is human readable and contains Java source code.

There are two ways to use the Log Wrapper:
  • Just after opening the 'real' repository (for example a Jackrabbit repository), wrap it using:
    {@link org.apache.jackrabbit.jcrlog.RepositoryLogger#wrap(javax.jcr.Repository, java.lang.String) RepositoryLogger.wrap(theRepository, settings)}.
  • Instantiate the repository using:
    RepositoryFactory.open(url).
    {@link org.apache.jackrabbit.jcrlog.RepositoryFactory#open(java.lang.String) RepositoryFactory.open(url)}.
Afterwards, the repository can be used in the usual way. All other classes are not meant to be used by the application directly.
